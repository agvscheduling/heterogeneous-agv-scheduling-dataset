# Heterogeneous AGV Instances

This repository stores experimental instances for collision-free scheduling of heterogeneous AGVs with speed adjustment.


## Dataset

- Grid layout: 30 instances
- Nishi layout: 30 instances
- Each instance contains 15 AGVs.
- Each AGV record contains an origin node, speed, and an ordered task node sequence.
- Objective function in the source files: Sum_arrival_time
